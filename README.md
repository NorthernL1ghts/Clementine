# Clementine

![Clementine Logo](Resources/Clementine-LOGO.png)

Clementine is a flexible and maintainable application framework built with a comprehensive application structure. It supports features such as:

- Command line argument parsing
- Performance timing
- Main thread management
- Hotkey termination
- Comprehensive core initialization and shutdown logic

## Features

- **Application Specification**: Includes details such as Name, Version, Width, Height, Author, and Contact.
- **Performance Timers**: Measure main thread work and wait times.
- **Application Management**: Create, run, and manage application lifecycle with robust threading support.
- **Hotkey Termination**: Terminate the application gracefully with a specified hotkey.
